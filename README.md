# Angular1 Tutorial

Project for learning purposes

## Install NodeJS:
```
    - curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
    - sudo apt-get install -y nodejs
    - npm install express
    - npm install --save body-parser
    - npm install --save nedb
```
## Initialize Project
```
    - npm start
```
